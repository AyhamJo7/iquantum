# Support

Thanks for using **iQuantum** — an open-source AI coding agent you run from your terminal.

## Getting help

- **Usage questions & ideas** — open a [GitHub Discussion](https://github.com/AyhamJo7/iquantum/discussions) (or an issue if Discussions is not enabled).
- **Bugs** — open a [GitHub Issue](https://github.com/AyhamJo7/iquantum/issues) using the bug template. Include your OS, Node/runtime version, iQuantum version, and steps to reproduce.
- **Feature requests / tasks** — use the feature request or task issue templates.

## Before opening an issue

1. Check the [README](README.md) and existing [documentation](README.md).
2. Search [existing issues](https://github.com/AyhamJo7/iquantum/issues?q=) — including closed ones.
3. Confirm you are on the latest release.

## Security

Please do **not** report security vulnerabilities in public issues. See [SECURITY.md](SECURITY.md) for private disclosure.

## Response expectations

iQuantum is maintained on a best-effort basis. Issues and PRs are triaged as capacity allows.
